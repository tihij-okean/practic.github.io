# Учебная практика — сайт курса

Статический сайт (HTML/CSS/JS). 

## Структура
- `index.html` — главная (о курсе, оценивание, календарь, витрина тем, шаблоны)
- `lessons/lesson1.html` … `lessons/lesson8.html` — страницы занятий
- `downloads/` — шаблоны для скачивания (DOCX/XLSX/PPTX)
- `assets/` — изображения и иконки
- `styles.css` — общий стиль
- `js/` — скрипты (витрина тем, чек‑листы прогресса)



Чек‑листы на страницах занятий сохраняются в `localStorage` браузера.
